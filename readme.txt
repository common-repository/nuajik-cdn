Speed up your WordPress with nuajik CDN.

== Description ==
The nuajik CDN speed up your website by caching all static content and serving them from our CDN all over the world.

Our CDN allows you to :

 - Accelerate the display of your website
 - Save bandwidth at your hosting provider
 - Better scalling on Web load


All our presence point have been selected to enable us to have the best network points of presence, to offer you greater responsiveness.
= Requirements =
You need to have a [nuajik account](https://admin.nuajik.io) & API access key to use this plugin. You can view the different CDN plan on your [website](https://www.nuajik.io/).
After creating your account, you have 14 days to test our CDN for free.

== Installation ==
1. You can either install this plugin from the WordPress Plugin Directory,
  or manually  [download the plugin]() and upload it through the 'Plugins > Add New' menu in WordPress
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Register at https://admin.nuajik.io to get an API key and create a slice associated with your wordpress instance.
1. Copy your API Key into the "API token" input in the plugin's settings
1. Select the right slice

== How to Use ==
Once you have selected the right slice in the plugin, everything will functions automatically
